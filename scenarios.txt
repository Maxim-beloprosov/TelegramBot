Здесь описаны все сценарии бота

Сценарий 1:
1. /start
1.1 Хочу посмотреть фильм
1.1.1 Выбираем рандомный фильм
- Отдаем пользователю рандомный фильм
/end

Сценарий 2:
1. /start
1.1 Хочу посмотреть фильм
1.1.2 Выбираем пожелания для фильма
1.1.2.1 Выбор жанра фильма
- Пользователь выбирает жанр из предложенных
- Запоминаем жанр
1.1.2.1.1 Рекомендация от конкретного пользователя
1.1.2.1.1.1 Выбор рекомендателя из возможных
- Отдаем пользователю фильм с учетом жанра и рекомендателя
/end

Сценарий 3 :
1. /start
1.1 Хочу посмотреть фильм
1.1.2 Выбираем пожелания для фильма
1.1.2.1 Выбор жанра фильма
- Пользователь выбирает жанр из предложенных
- Запоминаем жанр
1.1.2.1.1 Рекомендация от конкретного пользователя
1.1.2.1.1.1 Нет нужного пользователя в списке
- Отдаем пользователю фильм с учетом жанра
/end

Сценарий 4 :
1. /start
1.1 Хочу посмотреть фильм
1.1.2 Выбираем пожелания для фильма
1.1.2.1 Выбор жанра фильма
- Пользователь выбирает жанр из предложенных
- Запоминаем жанр
1.1.2.1.2 Рекомендация от любого пользователя с учетом жанра
- Отдаем пользователю фильм с учетом жанра
/end

Сценарий 5 :
1. /start
1.2 Хочу посоветовать фильм
1.2.1 Выбор жанра фильма
- Запоминаем жанр
1.2.1.1 Достаточно указать 1 жанр
1.2.1.1.1 Пользователь указывает название фильма
- Записываем данные о фильме в базу данных, если этот фильм еще не рекомендовали
/end

Сценарий 5 :
1. /start
1.2 Хочу посоветовать фильм
1.2.1 Выбор жанра фильма
- Запоминаем жанр
1.2.1.2 Нужно добавить еще 1 жанр
- Отдаем пользователю список из жанров, за исключением того, который он уже выбирал
1.2.1.2.1 Выбор 2ого жанра фильма
- Запоминаем 2ой жанр
1.2.1.2.1.1 Пользователь указывает название фильма
- Записываем данные о фильме в базу данных, если этот фильм еще не рекомендовали
/end
